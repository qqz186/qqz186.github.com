Vim 自学笔记

2011-2-5

第一部分：模式 Modes

Vim有三种模式

  • Normal Mode/Command mode：命令模式
  • Insert mode：大部分键入的是文本格式的。
  • Visual mode：帮助选择文本，也可以看成是命令模式的一种模式

从插入模式和可视模式进入命令模式，直接按<esc>

  • 如何进入插入模式（Insert Mode）
    • i insert 在光标前插入
    • I Insert 在光标所在整行前插入
    • a append 在光标后追加
    • A Append 在光标所在整行后追加
    • o open a new line below 在下面打开一行
    • O Open a new line above 在上面添加一行
  • 如何进入可视模式（Visual Mode）
    • v 以字符为单位选择
    • V 以行为单位选择

第二部分：移动Movement

基本的移动

  • h 左
  • j 下
  • k 上
  • l 右
  • Ctrl f PageDown 下一页翻页
  • Ctrl b PageUp 上一页翻页

进阶的游标移动

  • 0 数字0 移动到行首
  • ^ （摁shift+6）
  • $ 移至行尾以上两个按键源自正则表达式
  • G 移至文档末尾
  • gg 移至文档首部 （goto）
  • w 移至下一个word
  • W
  • e 移至前一个字字尾
  • E
  • b 移至前一个字字首
  • B
  • H 移至屏幕顶
  • M 移至屏幕中间
  • L 移至屏幕底部

特殊的移动

  • ( 上一个句子
  • )
  • { 上一个段落
  • }
  • % 匹配{}[]()只用

第三部分:Text Object

  • iw inner word
  • aw a word
  • iW inner WORD
  • aW a WORD
  • is inner sentence
  • as a sentence
  • ip inner paragraph
  • ap a paragraph
  • i(or i) inner block
  • a(or a) a block
  • i<or i> inner block
  • a<or a> a block
  • i{or i} inner block
  • a{or a} inner block
  • i" inner block
  • a" inner block
  • i` inner block
  • a` a block

Undo 和 Redo

  • u 撤销
  • Ctrl + R 撤销撤销
